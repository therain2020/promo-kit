# 配图设计规范

promo-kit 模板的设计规范。所有模板遵循 Dark Editorial Tech 设计系统。

---

## 设计原则

1. **Dark Editorial** — 深色背景 + 衬线大标题 + 霓虹点缀，营造"技术杂志"质感
2. **一图一意** — 每张图只传达 1-3 个信息点，不堆砌
3. **品牌一致** — 同套模板保持配色和风格统一
4. **可读性第一** — 文字清晰可读，对比度足够
5. **3:4 比例** — 小红书平台所有配图固定 3:4 宽高比（1080×1440 或 750×1000），禁止其他比例
6. **不依赖外部资源** — 所有 CSS 内联，优先使用系统字体栈

## 字体

```css
/* 展示标题（封面、大标题） — 衬线，营造编辑感 */
font-family: Georgia, "Noto Serif CJK SC", "STSong", "SimSun", serif;

/* 正文 — 中文无衬线 */
font-family: "PingFang SC", "Microsoft YaHei", -apple-system, BlinkMacSystemFont, sans-serif;

/* 等宽（代码、终端、编号） */
font-family: "JetBrains Mono", "Cascadia Code", "SF Mono", "Menlo", "Consolas", monospace;
```

| 元素 | 字号 | 字重 | 字体族 |
|------|------|------|--------|
| 封面大标题 (1080w) | 100-108px | 700 | serif |
| 封面大标题 (750w) | 42-44px | 700 | serif |
| 页面标题 | 60-72px | 700 | serif |
| 卡片标题 | 36px | 700 | sans |
| 正文 | 28-34px | 400 | sans |
| 代码 | 23-24px | 400 | mono |
| CTA 按钮文字 | 32-38px | 700 | sans |
| 顶部标签 | 22-26px | 500 | mono |

## 纹理与细节

所有深色模板使用 CSS-only 噪点纹理叠加（SVG 滤镜），避免纯黑的"空洞感"：

```css
body::before {
  content:""; position:absolute; inset:0; opacity:0.04;
  background-image: url("data:image/svg+xml,...");
  background-size: 256px 256px;
}
```

分辨率较高（1080×1440）的模板优先启用纹理。750×1000 的模板可省略。

## 排版规范

### 安全区域

- 上下各留 100px（1080w）或 80px（750w）
- 左右各留 88px（1080w）或 48px（750w）
- 标题放在上半部分

### 间距

| 元素间距 | 值 |
|---------|-----|
| 标题与副标题 | 28-36px |
| 段落之间 | 48-64px |
| 卡片之间 | 22-28px |

### 卡片样式

```css
.card {
  background: #131418;
  border: 1px solid #22242B;
  border-radius: 18px;
  padding: 36px 40px;
}
```

## 模板设计要点

### cover-xhs（小红书封面·深色编辑风）

- 1080×1440，衬线大标题 100-108px
- 上方 mono 标签（项目名），左侧紫色竖线 tagline
- 右下角大 emoji 作为视觉锚点
- 底部 GitHub CTA
- CSS 噪点纹理

### xhs-cover-tech（小红书封面·备用布局）

- 1080×1440，衬线大标题 108px
- CSS 网格线叠加
- 底部 pills 标签

### xhs-pain（痛点分析）

- 深黑底 + 红左边框卡片
- 大 emoji + 衬线标题
- 每项痛点独立卡片

### xhs-features（功能亮点）

- 深黑底 + 渐变蓝紫编号
- 每项功能独立卡片，带编号和标题
- 28px 描述文字

### xhs-steps（步骤教程）

- 深黑底 + 琥珀色步骤圆圈
- 竖线连接步骤
- 可选代码块（深底 + 边框）

### xhs-cta / cta-card（行动号召）

- 深黑底 + 径向辉光
- 蓝紫渐变按钮 + box-shadow 发光
- 居中布局，mono URL

### terminal / code-block（终端与代码）

- 1080×1440（XHS）或灵活
- 三圆点标题栏（#13161B）
- 自定义语法着色（紫 prompt、琥珀 arg、青 flag、绿 output）

### architecture（架构图）

- 深黑底 + 方框 + 箭头
- 高亮框：青色边框或紫色边框
